# 📊 Data Analytics & Business Intelligence Portfolio

مرحباً بك في مستودع مشاريع تحليل البيانات الخاص بي. يحتوي هذا المستودع على مجموعة من لوحات التحكم (Dashboards) والتقارير التفاعلية التي قمت ببنائها لتحليل بيانات متنوعة (مالية، موارد بشرية، مبيعات، وصحية) وتحويل البيانات الخام إلى رؤى قابلة للتنفيذ تدعم اتخاذ القرار.

---

## 🛠️ الأدوات والتقنيات المستخدمة (Tech Stack)
* **Business Intelligence:** Power BI (DAX, Power Query)
* **Data Analysis & Cleaning:** Microsoft Excel / SQL Server
* **Reporting:** PDF Reports & Interactive Dashboards

* **الوصف:** لوحة تحكم تفاعلية متقدمة لدراسة وتحليل مدى تأثير استخدام أدوات الذكاء الاصطناعي على الأداء الأكاديمي ومعدلات الطلاب (GPA) عبر التخصصات المختلفة.

  * **مقارنة الأداء الأكاديمي:** تحليل عميق للمعدل التراكمي قبل وبعد استخدام الذكاء الاصطناعي (Pre vs Post GPA) لمعرفة الأثر الحقيقي على التحصيل الدراسي لكل فئة من فئات الطلاب (Freshman, Senior, etc.).
